# 三十六计 · 技能五子棋

> 五子棋遇上三十六计，当棋盘上的博弈变成兵法谋略的较量。

打开 `gomoku.html` 即可开战。无需安装任何依赖。

## 玩法

- **15x15 棋盘**，标准五子棋规则，先连成五子获胜
- **技能系统**：每局随机抽取 6-10 个计策，双方共享技能池，- **能量系统**：每次落子获得能量，消耗能量使用计策
- **冷却时间**：每个计策使用后进入冷却，合理规划使用时机
- **AI 对手**：内置强力 AI，可与电脑切磋
